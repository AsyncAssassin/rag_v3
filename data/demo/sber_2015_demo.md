# Сбер: демо-выжимка 2015

Это синтетический демонстрационный документ для локального RAG-стенда.

Ключевые темы в 2015 году:
- рост эффективности операционных процессов;
- развитие цифровых каналов обслуживания;
- улучшение клиентского опыта;
- фокус на устойчивости и контроле рисков.

Контекст предназначен только для учебной демонстрации retrieval и generation.
